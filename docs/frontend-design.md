# 前端页面设计说明

## 用户角色

| 角色 | 可访问页面 |
|------|-----------|
| **教师** | 仪表盘、课程管理、智能答疑、作业管理（创建/批改）、学情看板、Agent构建器、平台配置 |
| **学生** | 仪表盘、智能答疑、作业管理（提交/查看反馈）、练习中心 |

## 嵌入式Widget（iframe模式）

供超星/钉钉嵌入的轻量独立组件，不带侧边栏，只包含核心功能。

```
┌────────────────────┐
│ EduAI 智能助手  [—] │
│────────────────────│
│                    │
│ 🤖 有什么可以帮你？│
│                    │
│ 👤 解释一下递归     │
│                    │
│ 🤖 递归是指函数    │
│ 调用自身的编程      │
│ 技巧...            │
│                    │
│ ┌──────────┐ [发送]│
│ │ 输入问题.. │      │
│ └──────────┘      │
└────────────────────┘
```

尺寸约 400x600px，通过URL参数传入课程ID和认证Token。

## 技术实现要点

| 页面 | 关键技术 |
|------|----------|
| 智能答疑 | SSE流式输出、react-markdown、react-syntax-highlighter |
| 批改结果 | 自定义批注组件（文本高亮+气泡定位）|
| 学情看板 | ECharts（雷达图、热力图、折线图）|
| 练习中心 | 表单交互、倒计时、答案对比渲染 |
| Agent构建器 | react-flow（拖拽画布、自定义节点、连线） |
| Widget | 独立路由、URL参数认证、响应式尺寸 |
