第 6 页：技术成果展示

标题：

技术成果展示

建议内容：

系统当前已完成如下主流程：

上传课程资料  
-> 后端异步处理  
-> 文本解析 / 分块 / 向量入库  
-> 用户提问  
-> 检索相关课程知识片段  
-> QAAgent 组织上下文  
-> 返回课程知识问答结果

配一句说明：

系统会优先检索与问题最相关的课程知识片段，再由 QAAgent 基于这些资料生成回答，因此问答结果能够体现课程知识约束，而不是通用大模型回答。
